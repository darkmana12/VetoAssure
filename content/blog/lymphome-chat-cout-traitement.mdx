## Le lymphome félin en quelques chiffres

Le lymphome est une **prolifération maligne des lymphocytes** (cellules immunitaires) qui peut toucher l'intestin, le médiastin (thorax), les ganglions, le rein, la peau ou le système nerveux. Selon le [Royal Veterinary College](https://www.rvc.ac.uk/), c'est le cancer le plus fréquent chez le chat domestique.

Quatre formes principales :
- **Lymphome digestif** (60% des cas) — chat de 8-12 ans, perte de poids et vomissements
- **Lymphome médiastinal** (15-20%) — chat jeune, souvent FeLV+, difficulté respiratoire
- **Lymphome multicentrique** — atteinte ganglionnaire généralisée
- **Lymphome extranodal** — peau, rein, œil, système nerveux

**Coût total moyen** : 4 000 à 6 500 € sur 6-12 mois pour un protocole complet. Sur lymphome digestif chronique avec rechutes, le coût cumulé peut atteindre **10 000 € sur 2-3 ans**.

## Pronostic selon le type

| Type | Médiane survie avec traitement | Médiane survie sans traitement |
|---|---|---|
| **Digestif petites cellules** (60%) | **2-3 ans** | 6-12 mois |
| **Digestif grandes cellules** | 6-12 mois | 4-8 semaines |
| **Médiastinal** (jeune chat, FeLV+) | 3-9 mois | 4-6 semaines |
| **Multicentrique** | 6-9 mois | 4-8 semaines |
| **Extranodal** (variable selon site) | Très variable | Très variable |

Le **lymphome digestif à petites cellules** offre le meilleur pronostic — un chat traité peut atteindre quasi-normalement son espérance de vie. C'est précisément le type le plus fréquent.

## Symptômes : 6 signes à surveiller

1. **Perte de poids progressive inexpliquée** — souvent le premier signe
2. **Vomissements et/ou diarrhée chroniques** (lymphome digestif)
3. **Baisse d'appétit**
4. **Fatigue, isolement, baisse d'activité**
5. **Ganglions périphériques palpables** (rare chez le chat)
6. **Difficulté respiratoire ou toux** (lymphome médiastinal)

**Règle d'or** : chez un chat de plus de 7-8 ans, toute perte de poids inexpliquée justifie un bilan vétérinaire. Plus le diagnostic est précoce, meilleur le pronostic.

## Le lymphome se prévient-il ?

Trois leviers documentés :

1. **Vacciner contre le FeLV** dès l'âge de 8 semaines pour les chats à accès extérieur — le FeLV multiplie par 60 le risque de lymphome
2. **Tester systématiquement** un nouveau chat avant introduction dans un foyer existant (FeLV + FIV)
3. **Bilan annuel** chez le chat > 7 ans — détection précoce de toute perte de poids ou anomalie sanguine
